== Description ==

Simply activate this plugin to cause your WordPress front page to fade in, like google.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `fade-in-like-google` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does every page fade in? =

No, just your front page.
